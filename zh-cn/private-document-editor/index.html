无需上传的隐私文档编辑器

在浏览器中编辑 DOCX、XLSX、PPTX 和 CSV 文件，文件从打开到保存始终留在你的设备上，不会上传到任何服务器。

最高隐私保护——工作原理

ByBrowser 通过 WebAssembly 在浏览器内运行完整的 OnlyOffice 办公套件。当你打开文档时，文件由浏览器本地文件 API 直接读取，不会通过网络传输。编辑引擎、格式转换和文件保存全部在浏览器标签页内完成，无需任何后端服务器参与。

与 Google 文档或 Office 365 有什么区别

  • 无需上传 — Google 文档和 Office 365 会将你的文件发送到云端服务器，ByBrowser 不会。
  • 无需账号 — 无需登录、无需邮箱、无使用跟踪
  • 加载后无需联网 — 安装为 PWA 后，无网络也可完整编辑文档
  • 开源可审计 — 查看代码、自行部署，亲自验证隐私承诺

支持的文件格式

  • DOCX — Microsoft Word 文档
  • XLSX — Microsoft Excel 表格
  • PPTX — Microsoft PowerPoint 演示文稿
  • CSV — 逗号分隔值文件

常见问题

这款编辑器适合哪些用户？
适合所有处理敏感文件（法律、医疗、财务或个人文件）、且不希望将内容发送到第三方服务的用户。
我可以验证没有数据被上传吗？
可以。编辑时打开浏览器 DevTools 的 Network 标签，你会看到没有任何文档内容被传输。
真的完全免费吗？
是的。ByBrowser 免费且开源，采用 AGPL-3.0 协议，没有付费版或使用限制。
技术上是如何实现的？
OnlyOffice 文档引擎被编译为 WebAssembly，完全在你的浏览器标签页内运行。打开、渲染和保存文件的过程中不涉及任何服务器端进程。
