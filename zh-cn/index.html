直接在浏览器中编辑文档，无需上传

ByBrowser 是一款基于 OnlyOffice WASM 的浏览器端文档编辑器，支持 DOCX、XLSX、PPTX 和 CSV 格式。文件始终留在你的设备上，不上传到任何服务器，无需注册账号即可使用。

浏览器端 Office 工具

DOCX 编辑器 — 在浏览器中编辑 Word 文档，无需上传 XLSX 编辑器 — 在浏览器中编辑 Excel 表格 PPTX 编辑器 — 在浏览器中编辑 PowerPoint 演示文稿 CSV 编辑器 — 在浏览器中查看和编辑 CSV 文件 隐私文档编辑器 — 最高隐私保护，无服务器 OnlyOffice WASM — 无需 Document Server 的浏览器端 Office 引擎 嵌入文档编辑器 — 通过 iframe 集成到你的应用 自部署文档编辑器 — 在自己的服务器上部署

工作原理

ByBrowser 通过 WebAssembly 在浏览器内运行完整的 OnlyOffice 引擎。当你打开文档时，文件由浏览器本地文件 API 直接读取，不会通过网络传输。所有渲染、编辑和保存操作均在浏览器标签页内完成，无需任何后端服务器参与。

主要功能

  • 无需上传 — 文档始终留在你的设备上，不会发送到任何服务器
  • 多格式支持 — 一个编辑器支持 DOCX、XLSX、PPTX 和 CSV
  • 支持离线使用 — 安装为 PWA 后，无需网络即可编辑文档
  • 可嵌入集成 — 通过 iframe 和 postMessage API 集成到你的应用中
  • 开源免费 — AGPL-3.0 授权，可自行部署到任意静态托管平台
  • 多语言界面 — 支持中文和英文界面

适合哪些用户

ByBrowser 适合所有需要在不将文件发送到第三方服务器的情况下打开或编辑 Office 文档的用户。常见场景包括：编辑机密合同或报告、在未安装 Microsoft Office 的设备上打开 DOCX 文件、在共享或受限环境中查看 XLSX 表格，以及需要将轻量级文档编辑器嵌入自己 Web 应用的开发者。

常见问题

ByBrowser 会上传我的文档吗？
不会。所有文档处理完全在你的浏览器内通过 WebAssembly 运行，不会向任何服务器发送任何内容。
需要注册账号吗？
不需要。无需注册或登录，打开编辑器即可直接使用。
支持哪些文件格式？
完整支持 DOCX（Word）、XLSX（Excel）、PPTX（PowerPoint）和 CSV 的打开、编辑和保存。
可以离线使用吗？
可以。首次访问后，你可以从浏览器地址栏将 ByBrowser 安装为 PWA，之后无需联网即可使用。
源代码是否公开？
是的。ByBrowser 以 AGPL-3.0 协议开源，代码托管在 GitHub 上。
