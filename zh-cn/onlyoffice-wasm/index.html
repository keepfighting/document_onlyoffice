OnlyOffice WASM 文档编辑器

纯静态 Web 应用，通过 WebAssembly 在浏览器内运行完整的 OnlyOffice 文档编辑引擎——无需 Document Server，无需后端，无需上传。

什么是 OnlyOffice WASM

OnlyOffice 是功能完整的开源办公套件，通常需要专用的 Document Server 来处理文件。ByBrowser 将 OnlyOffice x2t 转换引擎编译为 WebAssembly（WASM），直接在浏览器中加载 web-apps 前端，无需任何服务器基础设施。最终结果是一个完全在静态 HTML 页面中运行的功能完整的 Office 编辑器。

架构说明

  • OnlyOffice web-apps 提供编辑器 UI（文档、表格、演示文稿编辑器）
  • x2t WASM 处理 OOXML 格式（DOCX、XLSX、PPTX）与 OnlyOffice 内部格式之间的转换
  • 无 Document Server — socket.io 后端被本地回退替代，编辑器以桌面编辑器模式加载
  • Service Worker 缓存资源，支持重复访问和 PWA 安装

适用场景

  • 希望嵌入文档编辑器但不想运行后端服务器的开发者
  • 需要隔离网络文档编辑方案的组织
  • 研究如何将 OnlyOffice 作为纯客户端应用运行的技术人员

常见问题

与 OnlyOffice Document Server 有什么区别？
OnlyOffice Document Server 在服务器上进行格式转换。ByBrowser 使用 WebAssembly 在浏览器内完成所有转换，无需服务器。
这是 OnlyOffice 官方产品吗？
不是。ByBrowser 是一个独立的开源项目，基于 OnlyOffice 的开源组件（web-apps、sdkjs、x2t-wasm）构建。
可以自行部署吗？
可以。构建输出是一个静态文件目录，可以部署到任意静态托管平台——Nginx、Cloudflare Pages、Vercel 或 Docker 容器。
