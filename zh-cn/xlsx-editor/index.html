无需上传的在线 XLSX 编辑器

在浏览器中直接编辑 Excel 表格，无需将 XLSX 文件上传到任何服务器。所有处理通过 WebAssembly 在本地完成。

主要功能

  • 直接从电脑打开 .xlsx 表格，无需上传
  • 编辑单元格、公式、图表和格式，与 Microsoft Excel 完全兼容
  • 从零开始创建新表格并保存为 XLSX
  • 支持多工作表和命名区域
  • 支持通过 ?src= 参数从 URL 加载 XLSX 文件

为什么不要上传你的表格

财务模型、工资数据、销售管道和个人预算等 XLSX 文件通常包含敏感信息。将这些文件上传到在线服务存在数据泄露风险。ByBrowser 的 XLSX 编辑器完全在浏览器内运行，服务器永远不会收到你的文件副本。

使用步骤

  1. 点击上传按钮或拖拽 .xlsx 文件到编辑器
  2. 表格通过 OnlyOffice 引擎在本地打开
  3. 编辑单元格、套用公式、格式化数据
  4. 将更新后的文件下载到本地

常见问题

不安装 Microsoft Excel 也能打开 XLSX 文件吗？
可以。ByBrowser 使用 OnlyOffice 表格引擎，完整支持公式，无需安装 Excel。
公式和图表会保留吗？
会。OnlyOffice 引擎在打开和保存 XLSX 文件时完整保留 Excel 兼容的公式、数据透视表和图表。
可以从零创建新表格吗？
可以。使用菜单中的「新建 Excel」按钮创建空白工作簿，立即开始编辑。
