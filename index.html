How It Works

ByBrowser runs the full OnlyOffice engine inside your browser using WebAssembly. When you open a document, the file never leaves your device — all rendering, editing, and saving happens locally. There is no backend server processing your documents, and no account is needed to get started.

Key Features

  • No upload required — your documents stay on your device at all times
  • Multi-format support — DOCX, XLSX, PPTX, and CSV in one editor
  • Works offline — install as a PWA and edit documents without an internet connection
  • Embeddable — integrate the editor into your own application via iframe and postMessage API
  • Open source — AGPL-3.0 licensed, self-hostable on any static host
  • Multi-language — interface available in English and Chinese

Who Is This For?

ByBrowser is ideal for anyone who needs to open or edit Office documents without sending them to a third-party server. Common use cases include editing confidential contracts or reports, opening DOCX files on a machine without Microsoft Office installed, reviewing XLSX spreadsheets in a shared or restricted environment, and developers who need to embed a lightweight document editor into a web application without managing a backend.

Frequently Asked Questions

Does ByBrowser upload my documents?
No. All document processing runs entirely inside your browser using WebAssembly. Nothing is sent to any server.
Do I need to create an account?
No account or sign-up is required. Open the editor and start working immediately.
Which file formats are supported?
DOCX (Word), XLSX (Excel), PPTX (PowerPoint), and CSV are fully supported for opening, editing, and saving.
Can I use ByBrowser offline?
Yes. After the first visit, you can install ByBrowser as a PWA from your browser's address bar and use it without an internet connection.
Is the source code available?
Yes. ByBrowser is open source under the AGPL-3.0 license and available on GitHub.
