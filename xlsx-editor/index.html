XLSX Editor in Your Browser

Edit Excel spreadsheets locally in your browser without uploading XLSX files to a document server. All processing happens in your browser via WebAssembly.

What You Can Do

  • Open .xlsx spreadsheets directly from your computer — no upload needed
  • Edit cells, formulas, charts, and formatting exactly as in Microsoft Excel
  • Create new spreadsheets from scratch and save as XLSX
  • Works with multi-sheet workbooks and named ranges
  • Load an XLSX file from a URL using the ?src= parameter

Why Avoid Uploading Your Spreadsheets?

Financial models, payroll data, sales pipelines, and personal budgets often contain sensitive information. Uploading these to an online service puts your data at risk. ByBrowser's XLSX editor runs entirely in your browser — your spreadsheet is processed by WebAssembly locally, with no server ever receiving a copy.

How to Open an Excel File

  1. Click the upload button or drag and drop your .xlsx file
  2. The spreadsheet opens using the OnlyOffice engine — fully local
  3. Edit cells, apply formulas, and format your data
  4. Download the updated file to your device

Frequently Asked Questions

Can I open XLSX files without Microsoft Excel?
Yes. ByBrowser uses OnlyOffice's spreadsheet engine to open and edit XLSX files with full formula support, no Excel required.
Are my formulas and charts preserved?
Yes. The OnlyOffice engine maintains Excel-compatible formulas, pivot tables, and charts when you open and save XLSX files.
Is the editor free to use?
Yes. ByBrowser is completely free with no account, no subscription, and no file size limits imposed by the service.
Can I create a new spreadsheet from scratch?
Yes. Use the "New Excel" button in the menu to create a blank XLSX workbook and start editing immediately.
