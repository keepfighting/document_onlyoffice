无需 Excel 打开 XLSX 文件

有人发给你一个 .xlsx 表格，但你没有 Excel — 或者不想为 365 付费？在浏览器中打开它，公式原样保留。 无需安装，文件不外传。

打开你的 XLSX →

XLSX 是 Microsoft Excel 的格式。这个编辑器用 OnlyOffice 引擎打开它，因此单元格公式、数字格式、颜色和多个工作表 都能完整保留 — 它是一个真正的表格，而不是只读表格。它通过 WebAssembly 在本地运行，文件始终留在你的设备上。

你可以做什么

  • 打开任何 .xlsx（以及 .xls、.csv）文件，公式完整保留
  • 编辑单元格，再重新下载为 XLSX 或 CSV
  • 无需 Microsoft Excel，也无需 365 订阅
  • 不外传 — 你的表格永远不会离开你的设备
  • 可作为可安装 App 离线使用
开源且可自托管。 基于 AGPL-3.0 协议 — 你可以核实文件确实没有被上传，或者运行你自己的副本： github.com/ranuts/document。

常见问题

我能在没有 Excel 的情况下打开 XLSX 文件吗？

可以。它直接在你的浏览器中打开 XLSX — 无需 Excel 或 365 订阅。

公式和格式会被保留吗？

会。OnlyOffice 引擎会保留公式、数字格式和样式。

我的表格会被上传吗？

不会。它通过 WebAssembly 在本地运行，永远不会离开你的设备。

我能把它导出为 CSV 吗？

可以。打开后重新下载为 XLSX 或 CSV。
